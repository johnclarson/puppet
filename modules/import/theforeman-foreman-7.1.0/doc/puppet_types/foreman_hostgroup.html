Resource Type: foreman_hostgroup

Defined in:
lib/puppet/type/foreman_hostgroup.rb
Providers:
rest_v2

Overview

foreman_hostgroup registers a hostgroup in foreman.

Properties

  • ensure (defaults to: present)

    The basic property that the resource should be in.

    Supported values:
    • present
    • absent
  • environment

    The name of the environment to set as default for the hostgroup.

  • locations

    The name of the locations for this hostgroup.

  • organizations

    The name of the organizations for this hostgroup.

  • parent

    The name of the parent hostgroup for this hostgroup.

  • puppet_ca

    The name of the puppet CA smartproxy to set as default for the hostgroup.

  • puppetclass

    Puppetclasses to associate with this hostgroup.

  • puppetmaster

    The name of the puppetmaster smartproxy to set as default for the hostgroup.

Parameters

  • base_url (defaults to: https://localhost)

    Foreman's base url.

  • consumer_key

    Foreman oauth consumer_key

  • consumer_secret

    Foreman oauth consumer_secret

  • effective_user (defaults to: admin)

    Foreman's effective user for the registration (usually admin).

  • name (namevar)

    The name of the hostgroup.

  • timeout (defaults to: 500)

    Timeout for HTTP(s) requests
